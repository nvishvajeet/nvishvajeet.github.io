What it does (7 chained statements):

  1. Start sshd + set automatic startup
  2. Open firewall :22 (if not already open)
  3. Create C:\Users\nebula\.ssh
  4. Install the Nebula-HQ public key in authorized_keys
  5. Lock down ACLs on both
  6. Print the listening :22 row → proves sshd is up
