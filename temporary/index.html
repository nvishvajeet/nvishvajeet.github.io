TR64 onboarding — one PowerShell line

Open Windows PowerShell (Admin not required) on the TR64 / PHYSICS-HPC box. Paste this:

Invoke-WebRequest -Method POST -Uri http://10.2.6.92:8089/ -InFile $HOME\.ssh\id_ed25519.pub

What it does: takes your local SSH public key file and uploads it to the mini's key receiver. Expect a 200 OK response with OK: NNN bytes received. After that, ssh nebula@10.2.6.92 works with no password.
