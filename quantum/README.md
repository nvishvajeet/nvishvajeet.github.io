# Quantum Science & Technology Group demo

This is a concise, static prototype for a proposed interdisciplinary MIT-WPU
group. The working name and all roles are provisional until the group approves
them.

## People hierarchy

1. Project Leadership — Project Coordinator/PI and Co-PI
2. Administrative & Institutional Support — senior institutional enablement
3. Member Faculty — participating faculty across departments
4. Adjunct Faculty — external or visiting contributors, to be confirmed

Scientific leadership and administrative support are intentionally separated.
For the initial demo, Dr. Balaji Patil is shown as “Administrative &
Institutional Advisor.”

## Editing

- `data.js` contains all research areas, role sections, and people.
- `index.html`, `people.html`, and `member.html` are stable presentation pages.
- `render.js` turns the data into cards and profiles and hides missing fields.
- The site shares the base visual system with the Photonics demo and adds its
  own quantum-specific color and layout layer in `style.css`.

No database, CMS, tracker, or external JavaScript framework is used.
