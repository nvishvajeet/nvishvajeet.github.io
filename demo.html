PRISM — Live Demo

PRISM is a LAN-first Flask sample-request and instrument workflow system I built for MIT-WPU's shared lab facility. Sequential approvals (finance → professor → operator), queue management, per-request attachments, SHA-256 audit chain. Single binary, SQLite, no build step. Current stable release: v1.4.5.
